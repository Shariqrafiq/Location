Browser Geolocation API

This page demonstrates how websites can request your precise location using the browser's built-in Geolocation API — with your explicit consent.

🔒 What happens when you click below:
  • Your browser will show an official permission prompt
  • You can allow or deny — nothing runs without your choice
  • If allowed, your GPS/network location is read locally in your browser
  • No data is sent to any server — everything stays on this page
How It Works
1
Page calls navigator.geolocation.getCurrentPosition()
2
Browser shows its native permission dialog — you decide
3
If granted, the browser returns latitude, longitude, altitude, and accuracy data
4
The page uses that data — here, just displayed. Real apps might center a map or find nearby places
📚 Educational Note: The Geolocation API is a W3C standard built into every modern browser. Accuracy depends on available signals — GPS gives ~5m precision outdoors, while Wi-Fi triangulation gives ~50–100m, and IP-based gives only city-level.
